Workout

This is responsible for handling functions related to workout resource

Attributes

  • duration: Duration of the workout in minutes (float)
  • equipment: Equipment required for the workout (e.g., treadmill, none) (string)
  • workout_id: Unique identifier for the workout (integer)
  • workout_intensity: Intensity level of the workout (e.g., low, medium, high) (string)
  • workout_name: Name of the workout (string)
  • workout_type: Type of the workout (string)

Workout Plan

This is responsible for handling functions related to workout plan resource

Attributes

  • duration: Total duration of the workout plan (float)
  • plan_name: Name of the workout plan (string)
  • playlist_id: ID of the playlist (integer)
  • user_id: ID of the user (integer)

User

This is responsible for handling functions related to user resource

Attributes

  • email: The user's email address (string)
  • height: The user's height in centimeters. (float)
  • password: The user's password. (string)
  • user_type: The type of user (admin or regular user). (string)
  • weight: The user's weight in kilograms. (float)

Song

This is responsible for handling functions related to song resource

Attributes

  • song_artist: Artist of the song (string)
  • song_duration: Duration of the song in minutes (float)
  • song_genre: Genre of the song (string)
  • song_id: Unique identifier for the song (integer)
  • song_name: Name of the song (string)

Playlist

This is responsible for handling functions related to playlist resource

Attributes

  • playlist_duration: Total duration of the playlist (float)
  • playlist_id: ID of the playlist (integer)
  • playlist_name: Name of the playlist (string)
  • songs_list: Array od songs (string)
