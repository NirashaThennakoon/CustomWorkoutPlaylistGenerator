custWorkoutPlaylistGen:collection

List all songs from the backend. Only accessible with X_API_key.

custWorkoutPlaylistGen:song-all

List all playlist_ids for a particular song from the backend. Only accessible with X_API_key.

custWorkoutPlaylistGen:item

Leads to a resource that is used for requesting song from the backend for the corresponding song_id. Only accessible with X_API_key.

custWorkoutPlaylistGen:edit

Describes how to update song with the given song_id Only accessible with X_API_key.

custWorkoutPlaylistGen:delete

Link to delete the song. Only accessible by the X_API_key.
