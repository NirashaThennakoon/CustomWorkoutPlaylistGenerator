Workout Plan

custWorkoutPlaylistGen:playlist

Request the playlist from the backend for the given playlist_id. Only accessible with X_API_key.

custWorkoutPlaylistGen:author

Request the user for a particular workout plan from the backend. Only accessible with X_API_key.

custWorkoutPlaylistGen:item

Leads to a resource that is used for requesting workout plan from the backend for the corresponding workout_plan_id. Only accessible with X_API_key.

custWorkoutPlaylistGen:edit

Describes how to update workout plan with the given workout_plan_id Only accessible with X_API_key.

custWorkoutPlaylistGen:delete

Link to delete the workout plan. Only accessible by the X_API_key.
